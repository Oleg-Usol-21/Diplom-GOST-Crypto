# Diplom — шифрование по ГОСТ

Программа для шифрования файлов и папок с использованием:
- Алгоритм «Кузнечик» (ГОСТ Р 34.12-2015)
- Алгоритм «Магма» (ГОСТ Р 34.12-2015)
- Хэш-функция «Стрибог» (ГОСТ Р 34.11-2012)
